## 시작하기

일일 업무 보고 시스템은 팀원들의 업무 현황을 기록하고 관리하는 종합 업무 관리 시스템입니다.

### 주요 기능
- 일일 업무 보고서 작성 및 관리
- 부서별/날짜별 보고서 조회
- AI 기반 자동 요약 생성
- 프로젝트 진행 상황 관리
- 개인별 성과 리포트

## 로그인

### 로그인 방법
1. 시스템 접속 시 로그인 화면이 표시됩니다
2. **사원명**과 **비밀번호**를 입력합니다
3. [로그인] 버튼을 클릭합니다

### 비밀번호 변경
1. 로그인 후 우측 상단의 사용자 메뉴를 클릭합니다
2. [비밀번호 변경]을 선택합니다
3. 현재 비밀번호와 새 비밀번호를 입력합니다

### 비밀번호 분실 시
- 관리자 또는 운영자에게 비밀번호 초기화를 요청하세요

## 일일 보고서 작성

### 보고서 작성 페이지 접근
1. 상단 메뉴에서 [보고서 작성]을 클릭합니다

### 보고서 작성 방법

#### 1단계: 기본 정보 입력
| 항목 | 설명 |
|------|------|
| 날짜 | 보고서 작성 날짜 선택 |
| 사원명 | 본인 이름 선택 (부서 기준으로 필터링됨) |
| 부서 | 소속 부서 선택 |

#### 2단계: 업무 내용 입력
| 항목 | 설명 | 필수 여부 |
|------|------|----------|
| 업무 개요 | 오늘 수행한 주요 업무 내용 | 필수 |
| 진행 목표 | 다음 진행할 업무 목표 | 선택 |
| 달성률 | 업무 달성률 (0~100%) | 필수 |
| 팀장 평가 | 팀장의 평가 의견 | 선택 |
| 비고 | 추가 메모 사항 | 선택 |

#### 3단계: 저장
- [저장] 버튼을 클릭하여 보고서를 저장합니다
- 같은 날짜에 여러 건의 업무를 작성할 수 있습니다

### 연차/휴가 등록
- 업무 개요에 "연차"를 입력하면 다른 필드 없이 저장 가능합니다

## 보고서 조회

### 보고서 목록 페이지 접근
1. 상단 메뉴에서 [보고서 목록]을 클릭합니다

### 필터링 기능
| 필터 | 설명 |
|------|------|
| 날짜 | 특정 날짜의 보고서만 조회 |
| 부서 | 특정 부서의 보고서만 조회 |
| 사원명 | 특정 사원의 보고서만 조회 |
| 업체 | 특정 업체의 보고서만 조회 (운영자 전용) |

### 보고서 상세 보기
- 목록에서 보고서를 클릭하면 상세 내용을 확인할 수 있습니다

### PDF 내보내기
1. 원하는 날짜와 필터를 설정합니다
2. [PDF 내보내기] 버튼을 클릭합니다
3. 부서별 요약이 포함된 PDF가 다운로드됩니다

## AI 요약 기능

부서별로 당일 보고서를 AI가 자동으로 요약해 줍니다.

### AI 요약 생성 방법
1. 보고서 목록 페이지에서 원하는 날짜를 선택합니다
2. 부서 구분 행 아래에 있는 [AI 자동생성] 버튼을 클릭합니다
3. AI가 해당 부서의 보고서를 분석하여 요약을 생성합니다

### 요약 직접 작성/수정
1. [직접작성] 또는 [수정] 버튼을 클릭합니다
2. 요약 내용을 직접 입력하거나 수정합니다
3. [저장] 버튼을 클릭합니다

### 권한 안내
- **운영자**: 모든 부서의 요약 생성 가능
- **관리자/사용자**: 본인 부서의 요약만 생성 가능

## 프로젝트 관리

### 프로젝트 목록 페이지 접근
1. 상단 메뉴에서 [프로젝트 관리]를 클릭합니다

### 프로젝트 등록
1. [새 프로젝트] 버튼을 클릭합니다
2. 프로젝트 정보를 입력합니다:

| 항목 | 설명 |
|------|------|
| 프로젝트명 | 프로젝트 이름 |
| 부서 | 담당 부서 |
| 담당자 | 프로젝트 담당자 |
| 목표 종료일 | 최초 계획된 종료일 |
| 수정 종료일 | 변경된 종료일 (선택) |
| 상태 | 진행중/완료/대기/보류/취소 |
| 진행률 | 현재 진행률 (0~100%) |
| 주요 이슈 | 프로젝트 진행 중 이슈 |
| 세부 진행상황 | 상세 진행 내용 |

### 프로젝트 수정/삭제
- 목록에서 프로젝트를 선택하여 수정하거나 삭제할 수 있습니다

### 상태별 표시
| 상태 | 색상 |
|------|------|
| 진행중 | 파란색 |
| 완료 | 녹색 |
| 대기 | 노란색 |
| 보류 | 주황색 |
| 취소 | 빨간색 |

## 개인 리포트

개인별 업무 성과를 분석한 리포트를 확인할 수 있습니다.

### 개인 리포트 페이지 접근
1. 상단 메뉴에서 [개인 리포트]를 클릭합니다

### 리포트 생성
1. 사원명을 선택합니다
2. 조회 기간을 설정합니다
3. [리포트 생성] 버튼을 클릭합니다

### 리포트 내용
- 총 보고서 수
- 평균 달성률
- 주요 성과
- 개선 사항
- 프로젝트별 업무 요약

### AI 분석 리포트
- [AI 분석] 버튼을 클릭하면 GPT가 개인 업무를 분석하여 구조화된 리포트를 생성합니다

### 사용자 (User)
| 기능 | 권한 |
|------|------|
| 보고서 작성 | O |
| 보고서 조회 | 자사 데이터만 |

| 프로젝트 조회 | 자사 데이터만 |

| 사원 관리 | X |
| 업체 관리 | X |

### 관리자 (Manager)
| 기능 | 권한 |
|------|------|
| 보고서 작성 | O |
| 보고서 조회 | 자사 데이터 |
| 보고서 삭제 | O |

| 개인 리포트 | O |
| 사원 관리 | 자사 사원 |
| 업체 관리 | X |

### 운영자 (Operator)
| 기능 | 권한 |
|------|------|
| 보고서 작성 | O |
| 보고서 조회 | 전체 업체 |

## 자주 묻는 질문 (FAQ)

### Q: 로그인이 안 됩니다.
**A:** 사원명과 비밀번호를 정확히 입력했는지 확인하세요. 비밀번호를 분실한 경우 관리자에게 초기화를 요청하세요.

### Q: 보고서가 조회되지 않습니다.
**A:** 필터 설정을 확인하세요. 날짜, 부서, 사원명 필터가 올바르게 설정되어 있는지 확인합니다.

### Q: AI 요약 생성 버튼이 비활성화되어 있습니다.
**A:** 해당 날짜에 보고서가 존재하는지 확인하세요. 또한 본인 부서가 아닌 경우 요약 생성이 제한될 수 있습니다.

### Q: PDF 내보내기가 작동하지 않습니다.
**A:** 브라우저의 팝업 차단 설정을 확인하세요. 팝업이 차단되어 있으면 다운로드가 진행되지 않을 수 있습니다.

### Q: 다른 부서의 보고서를 볼 수 없습니다.
**A:** 역할에 따라 조회 권한이 다릅니다. 전체 데이터 조회가 필요한 경우 운영자에게 문의하세요.
